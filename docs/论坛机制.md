# 论坛机制

## 反馈

- 帖子和回复使用独立反馈体系，不复用旧赞踩。
- 支持评价、取消、切换。
- 同类型再次评价表示取消。
- 不同类型评价表示切换。
- 帖子和回复各自维护反馈计数。

## 排序

- `hot` 为默认排序。
- `hot` 按 `replyCount` 降序、`viewCount` 降序、`createdAt` 降序排列。
- `latest` 按 `createdAt` 降序排列。

## 回复

- 回复最多两层：顶级回复和子回复。
- 子回复必须挂在顶级回复下。
- 前端会高亮 `@用户名` 提及。

## 浏览计数

- 帖子详情访问会进入浏览计数队列。
- 浏览计数由后端异步累加。
